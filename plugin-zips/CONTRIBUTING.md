# Contributing to Re:Earth Visualizer Plugin Hub

Thank you for your interest in contributing to the Re:Earth Visualizer Plugin Hub.  
This repository is a hub for discovering plugins built for Re:Earth Visualizer.  
You can contribute by adding your plugin to the list or by improving this repository.  

---

## 1. How to Add a Plugin  
Already created a plugin? Here’s how to add it to the Plugin Hub.  
1. **Search:** Ensure your plugin isn’t already listed in the README.  
2. **Fork:** Fork this repository.  
3. **Add:** Add your plugin to the `README.md` file in the appropriate section.
   - Format: ```[Plugin Name](Link) - Description```
   - Keep descriptions concise and clear (1-2 sentences).
5. **Pull Request:** Submit a Pull Request with a clear title  
    (e.g. `Add visualizer-time-series-plugin`).
    

---

## 2. About Listed Plugins

To keep the Plugin Hub clear and useful, listed plugins should meet the following criteria.  
- ✅ Built for **Re:Earth Visualizer**
- ✅ Public repository with a README
- ✅ Clear description of what the plugin does
- ✅ Basic usage or installation instructions
- ✅ License information is provided

In addition, the following are recommended for better discoverability.  
- ✅ The plugin name makes it clear that it is for Re:Earth Visualizer
- ✅ The name is short, simple, and descriptive
- ✅ Including words such as `visualizer`, `reearth`, or `plugin` is encouraged

Examples:  
- reearth-visualizer-time-series-plugin
- visualizer-layer-control-plugin

This Plugin Hub does not review or guarantee plugin behavior.  
Each plugin is maintained by its respective author.

# Re:Earth Visualizer Plugin Hub への貢献について

Re:Earth Visualizer Plugin Hub に興味を持っていただきありがとうございます。  
このリポジトリは、Re:Earth Visualizer 向けに作られたプラグインを見つけるためのハブです。  
プラグインを一覧に追加したり、このリポジトリ自体を改善することで貢献できます。  

---

## 1. プラグインの追加方法  
すでにプラグインを作成している場合は、次の手順で Plugin Hub に追加できます。  

1. **Search（確認）:** README に、同じプラグインがすでに掲載されていないか確認してください。  
2. **Fork:** このリポジトリを Fork してください。  
3. **Add:** `README.md` の適切なセクションに、プラグインを追加してください。  
   - 形式: ```[プラグイン名](リンク) - 説明```  
   - 説明は簡潔に、1〜2文程度で記載してください。  
5. **Pull Request:** 内容が分かるタイトルで Pull Request を作成してください。  
    （例：`Add visualizer-time-series-plugin`）  

---

## 2. 掲載されるプラグインについて

Plugin Hub を見やすく、使いやすい場所に保つため、  
掲載されるプラグインには、次の条件を満たしていることをお願いしています。  

- ✅ **Re:Earth Visualizer 向け**に作られていること  
- ✅ README がある公開リポジトリであること  
- ✅ プラグインの機能が分かる説明があること  
- ✅ 基本的な使い方、またはインストール方法が記載されていること  
- ✅ ライセンス情報が明示されていること  

また、必須ではありませんが、次の点もおすすめしています。  

- ✅ 名前から Re:Earth Visualizer 用のプラグインであることが分かる  
- ✅ 短く、シンプルで、機能が想像しやすい名前になっている  
- ✅ `visualizer`、`reearth`、`plugin` といった単語が含まれている  

例:  
- reearth-visualizer-time-series-plugin  
- visualizer-layer-control-plugin  

本 Plugin Hub では、プラグインの内容確認や動作保証は行っていません。  
各プラグインは、それぞれの開発者によって管理・保守されています。
