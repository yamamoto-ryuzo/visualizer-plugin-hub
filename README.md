## Resources
### Plugin Development
Resources for developers who want to build plugins for Re:Earth Visualizer.
- [Plugin Development Guide](https://visualizer.developer.reearth.io/) - Official documentation for building plugins for Re:Earth Visualizer.
- [Plugin Playground](https://visualizer.reearth.io/plugin-playground?plugin-id=my-plugin) - A playground to test and preview your plugins directly in Re:Earth Visualizer.
- [Re:Earth Visualizer Plugin Template (React / Vite / ShadCN)](https://github.com/reearth-plugins/reearth-visualizer-plugin-shadcn-template) - A minimal template to develop a Re:Earth Visualizer Plugin using Vite, React, ShadCN, and Tailwind CSS.
