# Re:Earth Visualizer Plugin Marketplace(GitHub)

このリポジトリは、**Re:Earth Visualizer用プラグインのマーケットプレイス**です。

公式の[Re:Earth Marketplace](https://marketplace.reearth.io/)では、Re:Earth Visualizer用プラグインの提供が行われていないため、本リポジトリをRe:Earth Visualizer公式のマーケットプレイスとして運用しています。
ここには、各プラグインのリンクがまとまっています。


## 🙋‍♀️このリポジトリについて

- Re:Earth Visualizer用プラグインのリンク集です
- プラグイン本体は、各開発者・団体のGitHubリポジトリで管理されています
- 本リポジトリは、配布・動作保証・サポートは行いません

「どんなプラグインがあるかを探す場所」として利用してください。


## 📕利用方法（使う人向け）

1. 下の「プラグイン一覧」から気になるプラグインを探す
2. プラグイン名のリンクから、各プラグインのGitHubリポジトリへ移動
3. **利用方法・ダウンロード方法は、各プラグインのREADMEに従ってください**

## 👨‍💻プラグインを掲載したい方へ（開発者向け）

このマーケットプレイスへの掲載は歓迎しています。

掲載対象となるプラグインのリポジトリには、  
READMEに以下の内容を記載してください。

- プラグインの概要（何ができるか）
- Re:Earth Visualizer用プラグインであること
- 利用方法・インストール方法
- ライセンス情報
- （可能であれば）スクリーンショットやデモ

### 📗掲載手順

- このリポジトリをFork
- READMEの「プラグイン一覧」にリンクを追加
- Pull Requestを作成してください


## ⚠️注意事項

- プラグインの権利・責任は、各開発者に帰属します
- 各プラグインの利用条件は、それぞれのライセンスに従ってください
- 内容が不適切と判断された場合、掲載を取り下げることがあります
